# Dynamic Product Filter

## Course Outcome
**CO1** – Explain the fundamental concepts of front-end development using HTML, CSS, and JavaScript.

---

## 1. Aim
To create a dynamic product filter that sorts products based on user selection from a dropdown menu.

---

## 2. Objectives
- Implement filterable product data using JavaScript arrays  
- Create a sortable dropdown user interface  
- Display products using responsive product cards  
- Apply CSS Grid and animations for smooth layout transitions  

---

## 3. Hardware / Software Requirements

### Software
- Visual Studio Code  
- Web Browser (Google Chrome recommended)

### Hardware
- i3 or higher processor  
- Minimum 4 GB RAM  

---

## 4. About the Program
This experiment demonstrates dynamic UI updates using JavaScript array manipulation and DOM operations.  
Users can sort products by price using a dropdown menu.  
CSS Grid is used to create a responsive product layout, and hover animations improve user interaction.

---

## 5. Project Structure
dynamic-product-filter/
├── index.html
├── style.css
├── script.js
└── README.md
---

## 6. Technologies Used
- HTML5  
- CSS3 (Grid Layout & Animations)  
- JavaScript (ES6)

---

## 7. How to Run the Project
1. Download or clone the repository  
2. Open the folder in Visual Studio Code  
3. Open `index.html` in a browser  
   **OR** use the Live Server extension  

---

## 8. Output
The web page displays a list of products in card format.  
Users can sort products dynamically using the dropdown menu based on price (low to high or high to low).

---

## 9. Conclusion
The Dynamic Product Filter successfully demonstrates real-time data manipulation, responsive UI design, and interactive user experience using core web technologies.
